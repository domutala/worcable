# 🗺️ Worcable Roadmap

Worcable is evolving! Below is our vision for the future. Contributions are welcome for all phases.

---

### 🚀 Phase 1: Foundations & UX
- [x] Customizable recruitment pipelines.
- [ ] Advanced RBAC (Role-Based Access Control).
- [ ] CV Parsing & Management.
- [ ] Accessibility & Responsive UI overhaul.

### 🔔 Phase 2: Notifications & Integrations
- [ ] Automated email workflows.
- [ ] LinkedIn/Job boards integration.
- [ ] Webhooks (Slack/Teams).

### 🧠 Phase 3: Intelligence & Analytics
- [ ] Candidate scoring system.
- [ ] Advanced Dashboard (Conversion metrics).

### 🏢 Phase 4: Scalability & SaaS
- [ ] Multi-tenant support.
- [ ] Dockerized self-hosted SaaS version.
- [ ] Full Public API.

### 🌐 Phase 5: Community & Ecosystem
- [ ] Plugin/Extension architecture.
- [ ] Full Developer documentation.
- [ ] Open Source Marketplace for modules.
